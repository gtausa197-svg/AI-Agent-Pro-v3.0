## 🚀 Швидкий Старт

### Крок 1: Запустіть LM Studio Server

```bash
# 1. Відкрийте LM Studio
# 2. Завантажте вашу улюблену модель (напр., Mistral-7B, Llama-2-13B)
# 3. Запустіть сервер на http://localhost:1234
```

**Рекомендовані Моделі:**
- `mistralai/Mistral-7B-Instruct-v0.2` (Збалансована)
- `meta-llama/Llama-2-13b-chat-hf` (Висока якість)
- `TheBloke/openchat-3.5-GGUF` (Швидка)

### Крок 2: Налаштуйте Агента

Відредагуйте клас Config в `ai_agent.py`:

```python
class Config:
    # Кінцева точка LM Studio API
    LMSTUDIO_API_URL = "http://localhost:1234/v1/chat/completions"
    
    # Назва вашої завантаженої моделі
    MODEL_NAME = "mistralai/Mistral-7B-Instruct-v0.2"
    
    # Налаштування безпеки
    MAX_FILE_SIZE = 10 * 1024 * 1024  # 10 МБ
    FORBIDDEN_PATHS = [
        'C:\\Windows\\System32',
        'C:\\Windows\\SysWOW64',
        '/system', '/sys', '/proc'
    ]
```

### Крок 3: Запустіть Агента

```bash
# Стандартний запуск
python ai_agent.py

# АБО з GUI запускачем
python start_agent.py

# АБО з режимом налагодження
python ai_agent.py --debug
```

## 🐛 Усунення Проблем

### Проблема: Не Вдалося Підключитися до LM Studio

**Симптоми:**
```
ERROR: Не вдається підключитися до LM Studio на http://localhost:1234
```

**Рішення:**
```bash
# 1. Перевірте, чи працює LM Studio
curl http://localhost:1234/v1/models

# 2. Перевірте налаштування брандмауера
# Windows: Дозвольте Python через Windows Defender Firewall

# 3. Перевірте, чи завантажена модель
# Відкрийте LM Studio → Переконайтеся, що модель завантажена і сервер запущений

# 4. Перевірте доступність порту
netstat -an | findstr :1234
```

### Проблема: Модуль Не Знайдено

**Симптоми:**
```
ModuleNotFoundError: No module named 'psutil'
```

**Рішення:**
```bash
# Переустановіть залежності
pip install --upgrade -r requirements_extended.txt

# Або встановіть конкретно
pip install psutil requests
```

### Проблема: Доступ Заборонено

**Симптоми:**
```
PermissionError: [Errno 13] Permission denied: 'C:\\Windows\\System32'
```

**Рішення:**
```bash
# Це очікувано! Системні шляхи захищені за задумом.
# Використовуйте тільки дозволені директорії.

# Щоб змінити заборонені шляхи, відредагуйте Config.FORBIDDEN_PATHS
```

### Проблема: PIL/Pillow Не Працює

**Рішення:**
```bash
# Повне переустановлення
pip uninstall Pillow PIL
pip install --upgrade Pillow

# Перевірте встановлення
python -c "from PIL import Image; print(Image.__version__)"
```

### Проблема: База Даних Заблокована

**Симптоми:**
```
sqlite3.OperationalError: database is locked
```

**Рішення:**
```bash
# 1. Закрийте інші екземпляри агента
pkill -f ai_agent.py  # Linux/Mac
taskkill /F /IM python.exe  # Windows

# 2. Видаліть файл блокування (якщо існує)
rm knowledge_base/agent_memory.db-journal

# 3. Оптимізуйте базу даних
python -c "import sqlite3; conn = sqlite3.connect('knowledge_base/agent_memory.db'); conn.execute('VACUUM'); conn.close()"
```

### Проблема: Високе Використання Пам'яті

**Рішення:**
```bash
# 1. Зменшіть паралельні операції в Config
MAX_CONCURRENT_DOWNLOADS = 2

# 2. Періодично очищайте кеш
> auto_cleanup

# 3. Оптимізуйте базу даних
> optimize_database

# 4. Моніторте за допомогою
> monitor_performance 60
```
